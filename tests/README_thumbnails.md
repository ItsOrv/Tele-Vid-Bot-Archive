# ابزارهای کار با تامبنیل

این مستندات توضیحات لازم برای استفاده از ابزارهای کار با تامبنیل را ارائه می‌دهد.

## فهرست ابزارها

1. **اسکریپت تولید تامبنیل نمونه** (`generate_sample_thumbnails.py`): تولید تصاویر نمونه برای آزمایش
2. **اسکریپت تبدیل PPM به JPG** (`convert_ppm_to_jpg.py`): تبدیل تصاویر PPM به فرمت JPG
3. **آزمون‌گر URL** (`simple_url_tester.py`): تست اعتبارسنجی و شناسایی پلتفرم‌های مختلف

## اسکریپت تولید تامبنیل نمونه

این اسکریپت برای تولید تصاویر نمونه در دایرکتوری `tests/data/thumbnails` استفاده می‌شود. این تصاویر می‌توانند برای آزمایش و توسعه بیشتر ربات استفاده شوند.

### ویژگی‌ها

- تولید تصاویر رنگی ساده (قرمز و آبی)
- تولید تصاویر گرادیان با رنگ‌های مشخص و تصادفی
- ذخیره تصاویر در فرمت JPG با کیفیت بالا
- ایجاد فایل خلاصه JSON با اطلاعات تامبنیل‌های تولید شده

### نیازمندی‌ها

- پایتون 3.6 یا بالاتر
- کتابخانه PIL/Pillow (اختیاری، برای کیفیت بهتر تصاویر)

### نحوه استفاده

```bash
python tests/generate_sample_thumbnails.py
```

یا

```bash
chmod +x tests/generate_sample_thumbnails.py
./tests/generate_sample_thumbnails.py
```

### خروجی

اسکریپت چهار تصویر نمونه در دایرکتوری `tests/data/thumbnails` ایجاد می‌کند:
- `thumbnail_red.jpg`: تصویر قرمز ساده
- `thumbnail_blue.jpg`: تصویر آبی ساده
- `thumbnail_gradient.jpg`: تصویر گرادیان نارنجی به بنفش
- `thumbnail_gradient2.jpg`: تصویر گرادیان با رنگ‌های تصادفی

همچنین یک فایل `summary.json` حاوی اطلاعات تامبنیل‌های تولید شده ایجاد می‌شود.

## اسکریپت تبدیل PPM به JPG

این اسکریپت برای تبدیل تصاویر با فرمت PPM به فرمت JPG استفاده می‌شود. این تبدیل برای سازگاری بیشتر و کاهش حجم فایل‌ها انجام می‌شود.

### ویژگی‌ها

- تبدیل تمام فایل‌های PPM در دایرکتوری `tests/data/thumbnails` به JPG
- حذف فایل‌های PPM اصلی پس از تبدیل موفق
- نمایش گزارش تبدیل

### نیازمندی‌ها

- پایتون 3.6 یا بالاتر
- کتابخانه PIL/Pillow (اختیاری، برای کیفیت بهتر تصاویر)

### نحوه استفاده

```bash
python tests/convert_ppm_to_jpg.py
```

یا

```bash
chmod +x tests/convert_ppm_to_jpg.py
./tests/convert_ppm_to_jpg.py
```

## آزمون‌گر URL ساده

این اسکریپت برای آزمایش و اعتبارسنجی URL‌های مختلف استفاده می‌شود. می‌تواند پلتفرم‌های مختلف ویدیویی را شناسایی کند و اطلاعات مفیدی درباره URL ارائه دهد.

### ویژگی‌ها

- اعتبارسنجی URL
- تست اتصال و دریافت کد وضعیت HTTP
- شناسایی پلتفرم ویدیویی (یوتیوب، پورن‌هاب و غیره)
- استخراج شناسه ویدیو
- تولید URL‌های تلگرام
- ذخیره نتایج در فرمت JSON

### نیازمندی‌ها

- پایتون 3.6 یا بالاتر
- کتابخانه requests

### نحوه استفاده

```bash
python tests/simple_url_tester.py "https://example.com/video"
```

یا

```bash
chmod +x tests/simple_url_tester.py
./tests/simple_url_tester.py "https://example.com/video"
```

## مسیر خروجی

تمام خروجی‌های این اسکریپت‌ها در دایرکتوری‌های زیر ذخیره می‌شوند:

- تامبنیل‌ها: `tests/data/thumbnails/`
- نتایج آزمون URL: `tests/data/url_tests/`

## نکات

- برای بهترین نتیجه، کتابخانه PIL/Pillow را نصب کنید.
- اگر با خطای دسترسی مواجه شدید، اطمینان حاصل کنید که اسکریپت‌ها قابل اجرا هستند (`chmod +x`).
- برای تست URL‌های مختلف، می‌توانید از آزمون‌گر URL استفاده کنید.
